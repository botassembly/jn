=== Profile Structure ===

# Profiles stored at .jn/profiles/file/skills/

$ find .jn/profiles/file/skills -name *.json
.jn/profiles/file/skills/all-content.json
.jn/profiles/file/skills/catalog.json
.jn/profiles/file/skills/pdf-related.json
.jn/profiles/file/skills/web-related.json
# Example: skills/catalog.json
$ cat .jn/profiles/file/skills/catalog.json
{
  "pattern": "data/**/SKILL.md",
  "inject_meta": true,
  "filter": "select(.type == \"frontmatter\")",
  "description": "Skill catalog - frontmatter only from all SKILL.md files"
}
